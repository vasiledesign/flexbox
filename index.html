CSS

display: flex; / justify-content: center; align-items: baseline;

  • This will position the child item at the baseline of the parent container

  • If a child container's height is greater then the rest, all other child-items will be align with the bottom part at the same level

  • If you have text inside the child container that is aligned at the top, the baseline of alignment for these child-containers will be aligned according to this x-axis

CSS

display: flex; / justify-content: space-around; flex-wrap: wrap;

  • With the wrap property, child-elements will be wrapping around.

  • Resize browser to see this happen

  • You may also use flex-wrap: nowrap; this will force the child elements to remain on one line

  • You may also use flex-wrap: wrap-reverse; this will force the child elements to be displayed from the last to the first in reverse order
